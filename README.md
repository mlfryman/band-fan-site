Max Rebo Band Fan Site
=============

### About
Max Rebo Band Fan Site is a node.js web server app that details information about the amazing Max Rebo Band.

### Pages
```
Home
About
Discography
Tour Dates
```

### Features
- Express 
- Morgan
- EJS web engine
- HTML/CSS
